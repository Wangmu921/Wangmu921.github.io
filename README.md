# Wangmu921 的个人博客 Demo

这是一个适合部署到 GitHub Pages 的简单静态个人主页。

## 文件说明

- `index.html`：博客入口首页
- `styles.css`：页面样式
- `posts/hello-blog.html`：示例文章页

## GitHub Pages 部署

1. 在 GitHub 新建仓库：`Wangmu921.github.io`
2. 把这些文件上传到仓库根目录
3. 打开仓库的 `Settings` -> `Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main` 和 `/root`
6. 等待部署完成后访问：`https://Wangmu921.github.io`
